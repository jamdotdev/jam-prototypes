# Jam MCP Landing Page

A landing page prototype for the Jam MCP (Model Context Protocol) integration.

## Tech Stack

- **Next.js 14** - React framework
- **Radix Themes** - Component library with dark mode
- **TypeScript** - Type safety

## Features

- Dark mode design with purple/pink gradient accents
- Responsive layout
- Hero section with quick install command
- Features grid showcasing MCP capabilities
- Step-by-step installation guide
- Copy-to-clipboard functionality

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```
